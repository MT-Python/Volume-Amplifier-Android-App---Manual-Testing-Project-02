# 📱 Volume Amplifier Android App - Manual Testing Project

## Project Overview

This repository contains the complete **Manual Testing Documentation** for the **Volume Amplifier Android Application**. The project demonstrates the end-to-end manual testing process, including test planning, test scenario design, test case execution, bug reporting, and UI/UX improvement recommendations.

The objective of this project was to evaluate the application's functionality, usability, and overall user experience by identifying defects and providing actionable recommendations.

---

## Project Scope

The following features were tested:

* Volume Boost
* System Volume Control
* Equalizer
* EQ Presets
* Settings
* User Interface & Navigation
